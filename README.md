# 후쿠시마 잡화점의 기적
 기업사회캡스톤디자인(아이조아 팀)
 
## 프로젝트 소개
- 후쿠시마의 가상 상점 거리를 탐험할 수 있는 모바일 앱 개발
- 메타버스 맵을 통해 후쿠시마 제품을 소개하여 후쿠시마 제품에 대한 부정적 인식을 개선하고 지역 경제를 활성화하는 것이 목표

## 구현 부분
- 에셋 선정 (상점 거리, 판매 특산물, 캐릭터 등)
- 에셋을 맵에 불러와 배치
- Unity 내에서 3D 모델로 구현
- iOS 스마트폰이나 태블릿에서 조이스틱으로 캐릭터 이동
